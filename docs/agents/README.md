# Generated Agent Docs

This directory contains generated atlas views for the Agent Atlas repository itself.

Generate with:

```sh
atlas generate markdown
```

Expected generated files include:

```text
docs/agents/atlas.md
docs/agents/domains/*.md
docs/agents/workflows/*.md
docs/agents/components/*.md
docs/agents/documents/*.md
docs/agents/verification.md
```

Generated files should be compact routing docs, not a second codebase written in Markdown.
