# Agent Atlas CLI 0.16.0 Preview Release

Release artifact: `@agent-atlas/cli@0.16.0`

Dist tag: `preview`

## Compatibility

- Entity schema version: 1
- Registry config version: 1
- Usage receipt format: local `.agent-atlas/usage/` receipts generated by `atlas usage-note`
- Generated Markdown: current `atlas generate markdown` output from 0.16.0
- CLI arguments: all atlas-loading commands accept one positional root path or `--path <root>`, not both
- Boundary checks: public, private, and company profiles use the 0.16.0 boundary-check rules

## Public API Boundary

- Supported preview artifact: `@agent-atlas/cli`
- Internal packages bundled into the CLI are not stable public APIs:
  - `@agent-atlas/core`
  - `@agent-atlas/schema`
  - `@agent-atlas/markdown`
  - `@agent-atlas/mcp-server`

## Publish Notes

- First preview publish uses the `preview` dist-tag.
- Do not promote to `latest` until at least one public repository validates the published package.
- Public repos should pin the exact version: `pnpm dlx @agent-atlas/cli@0.16.0 ...`.
