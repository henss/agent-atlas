# Company Cross-Repo Sanitized Example

This example shows a central atlas registry for an organization with many repositories.

Everything is fictionalized. The point is the shape, not the names.

Useful commands:

```sh
node ../../packages/cli/dist/index.js global validate .
node ../../packages/cli/dist/index.js global manifest .
node ../../packages/cli/dist/index.js global generate markdown . --output docs/agents/global --check
```
